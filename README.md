# Kogel Numbers

This is a little script for finding 'Kogel numbers'.

A Kogel number is a number which value is the same as the index of its place in another number like Ï€.  
For example, in the number 67042, 4 is a Kogel number because it's the fourth digit.

The output will look like this:
```commandline
Shift : Number
1 : 1
1 : 16
2 : 45
1 : 61
1 : 75
```
